# JumpRush

A cyberpunk-themed endless runner game built with HTML, CSS, and JavaScript. Jump over obstacles, survive as long as you can, and beat your high score across 5 escalating speed levels.

🔗 **[Live Demo](https://elly-james.github.io/js-jumping-game/)**

## Features

- 5 levels with increasing obstacle speed
- Double jump unlocked at Level 3
- 3 lives per run — hit an obstacle and lose one
- Persistent best score saved across sessions
- Randomized obstacles with varying heights and shapes
- Pause, resume, and quit mid-run
- Animated cyberpunk visuals — scrolling city, starfield, glowing ground line
- Background music that changes per level, with mute toggle
- Score ticker and floating `+1` popups on each obstacle cleared
- Full keyboard and mobile tap support

## Controls

| Action | Keyboard | Mobile |
|---|---|---|
| Jump / Start | `Space` or `↑` | Tap the screen |
| Pause / Resume | `P` or `Esc` | — |
| Quit / Forfeit | `Q` | — |
| Mute / Unmute | `M` | — |
| Open guide | `H` button | — |

## Levels

| Level | Trigger | What changes |
|---|---|---|
| LV 1 | Start | Base speed |
| LV 2 | 200 pts | Speed increase |
| LV 3 | 400 pts | Double jump unlocked |
| LV 4 | 600 pts | Faster obstacles |
| LV 5 | 800 pts | Max speed |

## How to Run

- Clone the repository
- Open `index.html` in your browser

> Audio files are expected in an `audio/` folder (`level1.mp3` through `level5.mp3`). The game runs without them, music just won't play.
